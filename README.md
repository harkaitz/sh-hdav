# SH-HDAV

A minimalist DAV client that solely uses "curl(1)" and "xml2(1)".

## Help

hdav

    Usage: hdav ...
    
        -V              : Show configuration.
        -v              : Verbose mode.
        -p PATH         : Path to use.
        -u NEW=FILE ... : Upload files.
        -d FILE ...     : Download files.
        -o DIR          : Download directory.
        -l              : List files.
